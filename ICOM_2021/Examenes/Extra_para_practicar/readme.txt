Simulacro, potpurri y potpurri++ son cosas que deberian entrar para el primer
parcial, pero cambia año a año. Algunos ejercicios son adaptaciones de cuando
la materia era C, asi que sirven para practicar y pensar algoritmos pero puede
que ciertos temas de C++ no se toquen.

El resto deberian ser cosas para el segundo parcial y el final (entran los
mismos temas), pero tambien puede ser que algunos ejercicios se puedan hacer
con lo que se sabe hasta el primer parcial.
